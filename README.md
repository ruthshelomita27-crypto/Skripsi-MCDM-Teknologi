# Skripsi-MCDM-Teknologi
Repository ini berisi script Python yang digunakan dalam penelitian skripsi mengenai analisis kinerja keuangan perusahaan sektor teknologi di Bursa Efek Indonesia menggunakan metode Multi-Criteria Decision Making (MCDM).
## Metode yang Digunakan
- Entropy Weight Method
- TOPSIS
- VIKOR
- Spearman Rank Correlation
## Periode Penelitian
2020–2024
## Tools
- Python
- Google Colab
- Pandas
- NumPy
- SciPy
- Matplotlib
